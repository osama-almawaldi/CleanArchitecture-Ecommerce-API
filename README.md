CleanArchitecture-Ecommerce-API
هذا مشروع تخرج عبارة عن متجر إلكتروني متعدد المتاجر (Multi-Store)
بنيته باستخدام ASP.NET Core وطبقت فيه مبادئ Clean Architecture
كان هدفي الأساسي هو تنظيم الكود وفصل الطبقات عن بعضها عشان يكون النظام سهل للتطوير (Scalable) ومفهوم

التقنيات المستخدمة:
Framework: ASP.NET Core MVC / Web API.
Architecture: Clean Architecture.
Database: Entity Framework Core.
Patterns: Repository Pattern & Dependency Injection.

هيكلية المشروع:
قسمت الشغل لـ 4 طبقات رئيسية عشان الكود يكون نظيف:
Domain: للـ Entities والـ Interfaces.
Application: للـ Business Logic والـ DTOs.
Infrastructure: للتعامل مع قاعدة البيانات والـ Persistence.
Presentation: للـ Controllers والواجهات.
